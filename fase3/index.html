Propósito

Este sitio web tiene como propósito diseñar una estrategia para fomentar el aprendizaje y el reconocimiento del tema Flex Box, CSS Grid.

Flex Box

Flex es un nuevo valor HTML5 que permite realizar mockups o maquetas del sitio web de un forma más rápida y sencilla. Su nombre se debe al elemento que contiene la propiedad que recibe el nombre de display:flex. Permite especificar la alineación de los elementos y sus tamaños. Además, facilita la distribución del espacio restante, bien sea de manera horizontal o vertical, sin poner nada dentro de esos elementos. De esta manera, se pueden alinear los elementos que están dentro del flex, dando propiedades al mismo

CSS Grid

CCS Grid o rejilla es un sistema, como su nombre lo dice, para maquetear páginas web a través de rejilla, estandarizado y compatible con todos los navegadores. Su código es display:grid y es aplicable a la web para ordenadores de escritorio, para dispositivos con pantallas pequeñas y para el desarrollo de webapps. A diferencia de Flex Box, se aplica a dos dimensiones en vez de a una y esto hace que sea mucho más rápido y conveniente al momento de programar puesto que, entre otros aspectos, se "ensucia" menos el código CSS y no se recarga la página.
