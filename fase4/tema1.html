Flex Box: Diseño y comodidad

¿Para qué sirve?

El FlexBox o Flexible Box Layout permite realizar mockups de páginas web de manera más dinámica e inteligente, adaptando el diseño a la pantalla donde se muestra, manteniendo el concepto, desplazando o llenando los espacios con los elementos para que todo permanezca visible. Esto se logra gracias a que esta herramienta trabaja en 2 ejes: el principal que, por lo general, es horizontal, y el vertical. De este modo, se garantiza que el espacio que hay alrededor de estos elementos sea llenado de manera correcta.
Ejes de FlexBox. Recuperado de: https://ed.team/blog/guia-definitva-de-flexbox-1-main-axis-y-cross-axis

Fundamentos de CSS Flexbox

FlexBox se basa en un contenedor flexible que contiene varios elementos flexibles, llamados flex items. Cada eje sigue una dirección: mientras que el eje principal va de izquierda a derecha, el eje secundario va de arriba a abajo y de esta manera organizar los elementos en filas y columnas sin combinarlas.
