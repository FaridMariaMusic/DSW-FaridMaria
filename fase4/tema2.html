CSS Grid: Función Inteligente

¿Para qué sirve?

Al diseñar en la web, se debe disponer de un orden y una composición adecuada de los elementos para que el diseño sea óptimo, claro y atractivo. CSS grid layout no sólo permite diseñar de forma menos rudimentaria como HTML. sino que funciona para dispositivos móviles y es una herramienta responsiva. Al mismo tiempo, ofrece una rejilla para ubicar elementos en dos dimensiones, lo que permite un diseño más dinámico.

Fundamentos de CSS Grid

Grid Layout es un estándar y que no necesitas recargar tu página y tu código CSS para aplicarlo. No necesitas ensuciar tu HTML, ya que todo se aplica desde el CSS, por medio de nuevos atributos, valores y unidades. El sistema de rejillas es aplicable a la web para ordenadores de escritorio, para dispositivos con pantallas pequeñas y para el desarrollo de webapps.
